# sbtitool

sbtitool 是一个免费的在线 SBTI 测试工具，提供轻量化的网页答题体验、人格类型匹配、15 维度评分说明和结果海报展示。

## 网站地址

https://sbtitool.github.io/

## 项目说明

- 基于静态 HTML 页面构建，打开即可使用
- 提供 30 道核心题与补充题的在线测试流程
- 根据 15 个维度的得分情况匹配人格类型
- 展示结果解读、维度说明与对应图片资源
- 提供简体中文与繁体中文双页面，并支持语种切换
- 适合部署到 GitHub Pages 等静态托管平台

## 品牌说明

sbtitool 专注于打造清晰、轻量、易传播的在线测试体验。当前版本以趣味化测试和结果展示为主，适合日常浏览、分享和轻量娱乐使用。
